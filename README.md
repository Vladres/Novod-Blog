# Novod blog

## Version
* dotnet --version `Версия .NET Core`
* node --version   `Версия Node.js`
* npm --version    `Версия npm`
* ng --version     `Версия Angular CLI`
## Install packages
* `cd NovodBlog/NovodBlog/ClientApp` `npm install` ;

## Open in localhost
* `cd ..`
* `dotnet run`
